고객센터

  • 비밀번호를 틀리게 입력하면 어떻게 되나요?

    임시비밀번호 확인이 어려운 경우, [1:1문의] 또는 고객센터(☎ 080-710-1020)로 문의 주시면 확인 후 안내해 드리도록 하겠습니다.

  • 비밀번호 변경은 어떻게 하나요?

    비밀번호는 [MYPAGE>회원정보>회원정보수정] 메뉴에서 현재 비밀번호와 새로운 비밀번호를 입력하시면 변경 가능합니다.

  • 비밀번호가 기억나지 않아요

    프로스펙스닷컴 회원의 비밀번호는 아래의 방법으로 확인가능합니다.

    [등록한 이메일로 찾기]
    1. LOGIN > 아이디/비밀번호 찾기를 클릭
    2. FIND PASSWORD 탭 선택 > 비밀번호 찾기 방법 "등록된이메일" 선택
    3. 가입시 입력한 이메일아이디/이름/휴대폰번호를 입력하고 '확인'버튼을 클릭
    4. 수신된 이메일에서 인증 클릭 하시면 비밀번호 재설정이 가능합니다.

    [등록한 휴대폰으로 찾기]
    1. LOGIN > 아이디/비밀번호 찾기를 클릭
    2. FIND PASSWORD 탭 선택 > 비밀번호 찾기 방법 "등록된휴대폰" 선택
    3. 가입시 입력한 이메일아이디/이름/휴대폰번호를 입력하고 '확인'버튼을 클릭
    4. 휴대폰번호로 전송된 인증번호 입력 후 '확인' 버튼을 클릭하시면 비밀번호 재설정이 가능합니다.

    ※ 비밀번호 찾기가 원할하지 않는 경우에는 고객센터(☎ 080-710-1020) 로 문의해 주시기 바랍니다.

  • 아이디가 기억나지 않아요

    프로스펙스닷컴 회원의 아이디는 아래의 방법으로 확인가능합니다.

    1. LOGIN > 아이디/비밀번호 찾기를 클릭
    2. FIND ID 탭 선택 > 가입시 입력한 이름/생년월일/휴대폰번호를 입력하고 '인증번호받기'를 클릭
    3. 휴대폰번호로 전송된 인증번호 입력 후 '확인' 버튼을 클릭하시면 회원님의 아이디가 화면에 표시됩니다.

    ※ 아이디 찾기가 원할하지 않는 경우에는 고객센터(☎ 080-710-1020) 로 문의해 주시기 바랍니다.

  • 회원탈퇴가 되질 않아요.

    프로스펙스닷컴에서 현재 진행중인 거래 내역이 있는 경우 즉시 탈퇴가 불가능합니다.

    1. 진행 중인 거래가 있는 경우, 주문/배송 상태가 배송완료 이후 탈퇴가 가능합니다.
    2. 취소, 교환, 반품이 진행 중이라면 해당 진행이 완료 후 탈퇴가 가능합니다.
